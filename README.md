# Reva Challenge - Booking Calendar

This mobile application is developed using **React Native** and **Expo** for the Reva technical challenge. The app allows users to view and manage sports field bookings.

## Technologies Used
- **React Native / Expo** (SDK 51+)
- **TypeScript** for robust and clean code.
- **Axios** for API consumption.
- **Environment Variables** for API key security.

## Setup and Installation
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Create a `.env` file in the root directory with the following variables:
   ```text
   EXPO_PUBLIC_API_URL=[https://admin.reva.la/api/u](https://admin.reva.la/api/u)
   EXPO_PUBLIC_API_TOKEN=your_token_here
